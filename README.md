# E-Commerce Sales & Profitability Analysis

## Project Overview
This project analyzes e-commerce transaction data to understand product-level
revenue and profitability. The analysis was performed using MySQL for data
exploration and Power BI for dashboarding.

## Business Questions Answered
- Which products generate the highest revenue?
- Which products are most profitable?
- How does revenue differ across products?

## Key Metrics
- Total Revenue
- Total Profit

## Key Insights
- A small number of products contribute a large share of total revenue.
- High revenue does not always mean high profit due to product costs.
